### BRANCHES desse projeto
A branch desse projeto se refere a ser compativel com Linux, Mac e Windows, tirando a biblioteca <conio.h>(feita para windows)</br>
Se a repo principal nao funcionar considere usar o branch Projeto sem <conio.h> 

## Instalação:
>clone o repositorio para a pasta de sua preferencia


## Principal
```bash
git clone https://github.com/davimaires/Projeto-Estruturas-de-Dados/tree/main/pilha_torre_de_hanoi ~/Torre_de_Hanoi
cd ~/Torre_de_Hanoi
chmod +x main.c
./main.c
```
